# mk-tools

mk命令

## 安装

```
$ npm i -g mk-tools
```

## 创建空网站

```
$ mk website [website name，可空]
```

## 创建应用

```
$ mk app [app name, 可空]
```

## 克隆应用

```
$ mk clone [app name, npmjs发布的应用， 可空]
```

## 编译

```
$ mk compile
```

## 完整演示创建一个网站

```
$ npm i -g mk-tools
$ mk website www
$ cd www
$ mk clone mk-app-root root
$ mk compile
$ mk start
```
